# 🚀 Courses API (Node.js + Express)

A simple RESTful API built with **Node.js** and **Express.js** for managing courses.  
This project demonstrates CRUD operations with in-memory data storage and input validation.

---

## 📌 Features

- Create a new course
- Get all courses
- Get a single course by ID
- Update a course
- Delete a course
- Input validation using `express-validator`
- Clean architecture using controllers and routes

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- express-validator
- UUID
